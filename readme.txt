= Can I use a single NinjaChat code in multiple domains? =

Yes! You can add NinjaChat to as many sites as you want, and manage chats from one account.

However, we recommend you generate different live chat codes for each website, to avoid spammers. Since they might use your live chat code on their website to increase the chat volume on your site artificially.


= Why chat widget goes offline? =

The chat widget on your website helps you to connect with your visitors via chat. If the chat widget is hidden or grayed out, the visitor will not be able to reach you for help. 

= Do I need a developer to install NinjaChat? =

If you have access to your website HTML/source code, then you will not need a developer to help you. To install NinjaChat on your website, you simply have to copy and paste the code snippet we provide to your website's HTML page above the </body> tag.
